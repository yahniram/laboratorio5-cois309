Sección 2 – Párrafo con cambio de color

Esta página forma parte del Laboratorio 4 del curso Web Page I. El objetivo principal es aplicar diferentes elementos de HTML, CSS y JavaScript en una estructura organizada de múltiples secciones. A través de este diseño se practica el uso de tipografías variadas, tablas, videos y formularios. Además, se integra un comportamiento visual mediante el cambio de color al pasar el cursor sobre este párrafo. Finalmente, se refuerza el concepto de diseño responsivo para adaptar el contenido a diferentes tamaños de pantalla.
